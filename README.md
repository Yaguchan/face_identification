# face_identification
### データ加工(データ拡張用)
解像度を変更した画像を作成
```
python preprocess/data_resize.py
```
### 学習
学習に使用するデータとそのメンバーリストを与えて学習
```
python run.py
```
### 推論
ピンの写真の推論
```
python inference_sample.py
```
